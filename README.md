- Current Temperature
- Weather Condition
- Feels Like Temperature
- Wind Speed
- Humidity
- Pressure
- Visibility
- UV Index
- Air Quality (PM2.5, PM10, CO)
- Sunrise & Sunset
- 3-Day Forecast
- Rain Probability

# 🚀 How to Use

1. Download the `.pbix` file.
2. Open it in **Power BI Desktop**.
3. Replace the WeatherAPI key with your own (if required).
4. Refresh the dataset.
5. Explore the dashboard.
